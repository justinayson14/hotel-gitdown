# hotel-gitdown

Hotel project that handles hotel reservations for Hotel GitUp. Project for Comp 380: Intro to Software Engineering.

## Description

Hotel GitDown is a desktop application designed to manage hotel reservations. It provides a user-friendly interface for guests to check-in, book rooms, view room details, see amenities, and check-out. This project serves as a practical application of software engineering principles, built using Java and JavaFX.

## How to Run

To run this application, you will need to have a Java Development Kit (JDK) installed on your system.

1. **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/hotel-gitdown.git
    cd hotel-gitdown
    ```

2. **Run the application using the Maven wrapper:**
    - On Windows, run: `mvnw.cmd javafx:run`
    - On macOS/Linux, run: `./mvnw javafx:run`

## Technologies Used

- **Java**: Core programming language.
- **JavaFX**: GUI framework for the user interface.
- **FXML**: XML-based language for defining the structure of the user interface.
- **Maven**: Build automation and dependency management tool.
